# 📠 Sentiric Sip Cli

Sunucu tarafı SIP/RTP uygulamalarını stres ve mantık testlerine tabi tutmak için tasarlanmış otonom komut satırı aracı.

## 🚀 Hızlı Başlangıç
```bash
cargo run --release
```

## 🏛️ Mimari ve Mantık
* **Geliştirici Kuralları:** Gizli [.context.md](.context.md) dosyasını okuyun (AI Ajanları için zorunludur).
* **İş Mantığı ve Algoritmalar:** [LOGIC.md](LOGIC.md) dosyasını inceleyin.
* **Anayasal Konum:** [sentiric-spec/spec/tools/sip-cli.yaml](https://github.com/sentiric/sentiric-spec)
